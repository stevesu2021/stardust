## 🏗️ 技术架构

### 前端技术栈
```
Uni-app 3 (Vue 3 + TypeScript)
├── uView UI Plus (UI组件)
├── Pinia (状态管理)
├── pinia-plugin-persistedstate (持久化)
└── Vite (构建工具)
```

### 后端技术栈
```
Nest.js (TypeScript)
├── Prisma (ORM)
├── PostgreSQL (主数据库)
├── Redis (缓存/Session)
├── JWT + Passport (认证)
└── lunar-javascript (命理计算)
```

### 部署架构
```
Nginx (反向代理 + 静态资源)
├── CDN (全球加速)
├── Docker (容器化)
├── Kubernetes (容器编排，可选)
└── 监控 (Prometheus + Grafana)
```

## 🎯 快速开始

### 1. 环境准备
```bash
# 安装 Node.js 18+
node -v

# 安装 Docker
docker --version

# 安装依赖
npm run install:all
```

### 2. 本地开发
```bash
# 启动后端
cd backend
npm run start:dev

# 启动前端 (H5)
cd frontend
npm run dev:h5

# 启动前端 (小程序)
cd frontend
npm run dev:mp-weixin
```

### 3. 生产部署
```bash
# 构建并启动
docker-compose -f docker-compose.prod.yml up -d

# 查看日志
docker-compose logs -f
```
